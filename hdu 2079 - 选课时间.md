# 定位

 - 排列组合问题

 - 母函数

# 分析

 - 物品种类数 $k$，第 $i$ 行输入：物品价值 $v[i]$，物品数量 $n2[i]$。

 - 母函数 $$g(x)=(1+x^{v[1]}+...+x^{n2[1]v[1]})...(1+x^{v[i]}+...+x^{n2[i]v[i]})...(1+x^{v[k]}+...+x^{n2[k]]v[k]})$$
